⚡ Data Providers 3 sumber
Shinigami Default

Provider utama menggunakan REST API langsung. Data JSON terstruktur dengan dukungan status dari AniList GraphQL API. Retry otomatis 3x dengan delay 500ms.

REST API JSON Native AniList Status Auto Retry Cache 30min
Komiku Scraper

Provider alternatif dari komiku.org. Menggunakan HTML scraping dengan Cheerio. Mendukung 42+ genre dengan listing dan detail halaman.

HTML Scraping Cheerio Parser 42+ Genre Bahasa Indonesia
Kiryuu Hybrid

Provider hybrid yang menggabungkan HTML scraping dengan WordPress REST API. Search via HTMX POST, metadata via WP JSON endpoint.

WordPress API HTML + REST HTMX Search Batch Resolve
